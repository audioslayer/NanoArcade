# Claude Instructions

## Git Workflow

**IMPORTANT:** Do not commit and push changes unless explicitly instructed by the user. Always wait for user approval before running `git commit` and `git push`.

## Project Guidelines

- This is the NanoArcade Manager project - a box art manager for retro gaming handhelds
- Supports multiple devices: RG Nano, Miyoo Mini/Flip, Trimui Brick, MinUI, and NextUI
- Uses File System Access API for local file management
- Images stored in device-specific folders (Imgs, .res, .media, etc.)
